- DLL SYLVAC BT x.xx:  DLL to communicate with our instruments via the customer's own application
- TestDLLCSharp.zip : C# application test example

Content:
- DLL   : Sylvac communication DLL (Win7 and Win10) 
- API   : HTML documentation that describes the DLL methods (press Index.htm to start browsing)
- Header: Files needed to integrate the DLL in an application

- There is two DLL available. 
	- The Win10 DLL detects the native Bluetooth card and connect to it (works with dongle too). 
	- The Win7 DLL will only work with dongle because the library to communicate with the native Bluetooth is not available on win7.
